# degrees-radians
[![NPM version][npm-image]][npm-url]
[![build status][travis-image]][travis-url]
[![Test coverage][coveralls-image]][coveralls-url]

Convert degrees to radians.

## Installation
```bash
$ npm i --save degrees-radians
```
## Overview
```js
var radians = require('degrees-radians');

radians(90);
// => 1.5707963267948966
```
